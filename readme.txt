Add Interchangeable words that replace one another in a sentence through CSS transitions.
== Description ==
This plugin will allow you to add interchangeable words that will replace one another in a sentence after intervals.This would allow you to display your multiple content in just one sentence and attract user to your site.

Note: This plugin requires <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed" target="_blank">Visual Composer</a> Plugin.
